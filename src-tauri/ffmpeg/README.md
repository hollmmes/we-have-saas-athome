# FFmpeg Binaries

Bu klasöre FFmpeg binary dosyalarını eklemeniz gerekiyor.

## İndirme

Windows için FFmpeg'i buradan indirin:
https://github.com/BtbN/FFmpeg-Builds/releases

**İndirmeniz gereken dosya:**
`ffmpeg-master-latest-win64-gpl.zip`

## Kurulum

1. Zip dosyasını indirin
2. `bin` klasöründen şu dosyaları bu klasöre kopyalayın:
   - `ffmpeg.exe`
   - `ffprobe.exe`
   - `ffplay.exe` (opsiyonel)

## Notlar

- Bu dosyalar ~100MB olduğu için Git'e commit edilmez
- Build sırasında otomatik olarak uygulamaya paketlenir
- Kullanıcılar uygulamayı indirdiğinde FFmpeg dahil gelir
